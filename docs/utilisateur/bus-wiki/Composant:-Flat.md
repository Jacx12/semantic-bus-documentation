Transformer un tableau de tableaux en un unique tableau.

### Fonctionnement technique

Utilise la fonction javascript flat() sur un tableau de données en entrée.

### Exemple 

Si on a ces données en entrée : 
```json
["16","15","11","03","06","08","01","17","04","18","19"],["16","01","17","04"],["01","17","04"]
```

Le résultat ressemblera à ça en sortie : 
```json
["16","15","11","03","06","08","01","17","04","18","19","16","01","17","04","01","17","04"]
```
